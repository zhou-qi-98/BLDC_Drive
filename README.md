# 三相无刷直流电机驱动设计

#### 介绍

* 采用STM32F103C8T6作为主控
* 采用三相逆变桥电路
* 采集三相电流
* 前级采用栅极驱动MOS，自带闭锁
* 采集电机驱动电压，与MOS电流
